# Game Boy (DMG-01)

### Specs
  - CPU: 8 Bit 4.19 MHz
  - RAM: 8 KB
  - Screen type: LCD (no backlight)
  - Screen size: 47 x 43mm
  - Screen resolution: 160 x 144 pixels
  - Power: 4x AA batteries
  - Dimensions: 148 x 90 x 32 mm
  - Weight: 220g
